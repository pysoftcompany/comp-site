PySoft-s.r.o

PySoft-s.r.o is an independent Software and research company.
Our primary focus is on Python, which is one of the most versatile programming languages out there.


Using the power of Python and other bleeding edge technlogies PySoft-sro focuses mainly on Web Development, Network Programming, Security and other similar applications and software products.


Our servers run in cloud mainly world class Linux servers which we configure to optimize performance. We use various web server combinations to serve applications in a reliable way.

Our source code is managed using Git and processed through GitHub, Git-Lab or Bit-bucket depending on scope of the software. We run Continuous Integration to make sure our processes are going smoothly. As an agile company alongside writing the software we run simultaneously automated tests to find out pain points and resolve any issues in a timely manner. Software is packaged and distributed as a package and per request can be deployed using modern methods like ansible or fabric. As we walk the talk we have a metric system to monitor performance.

We are passionate about open source and use these technologies to provide the best service , as well as give back to open source community by running independent research and providing training sessions, contributions.
