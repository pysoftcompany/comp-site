Courses



@PySoft s.r.o. We not only produce Software products but we try to expand your thinking as well!

There are many if not unlimited challenges when it comes to self-paced online learning , one of them being you get stuck and it can be hard to get help to get yourself out of that specific problem and continue with your hacking. That is why @ PySoft s.r.o. we offer a Python programming and Introductory Networking courses which are hands-on and teach you not only programming but rather also tackling problems , critical thinking and software design patterns.

Can we get more details about classes?

A typical course day starts with coffee around 9:00am. The session runs from 9:30-5:30pm with a lunch and another coffee break. Trainning session is divided between ready theory and hacking on projects.There is time to code , to ask questions and to discuss.



Am I or the team is ready to take a course?

Courses are attended by folks with different backgrounds. Eventhough there are no specific prerequisites, You or your team should be comfortable with using basic utilities on a laptop computer.

Course Summary

Programming is fun and profitable. Acquring knowledge and becoming a software developer should also be in similar manner fun! Our course will teach you everything you need to know about the Python language all the while building interesting and engaging applications.
  • Setting up development environment with Python and a code editor(IDE)
  • Going through data types such as integers, strings, lists, and dictionaries
  • Looping to get a lot of work done with a little bit of effort(shorter codes)
  • Accessing Built-in Python modules and their functionalities
  • Code-reading(not cold reading ☺) , writing codes and running programs
  • Navigation of the command line to get ready for writing larger scripts
  • Creating our own functions to envelope useful work for future use
  • Using Object Oriented Programming via classes to blue-print to collect, name, and reuse functions and variables
  • Practicng what we have learned with projects
